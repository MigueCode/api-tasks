Hello !! Welcome to the tasks api

Here you can find a little and light api to use in your frontend projects.

How to start

Once you have your frontend and you need to make request you can do it from our api following the next steps

Here you have the endpoints for the diferent requests

POST Request

Endpoint: https://api-tasks-dev.herokuapp.com/api/tasks

GET Request

Endpoint: https://api-tasks-dev.herokuapp.com/api/tasks

DELETE Request

Endpoint: https://api-tasks-dev.herokuapp.com/api/tasks/param

Here in the delete method you need to specify the param in the url which is the id of your element to delete
